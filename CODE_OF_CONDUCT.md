
# Code of Conduct

This project is about writing good code and working together respectfully.

## Rules

1. Be respectful.
2. Don’t be a jerk.
3. Keep discussions technical and constructive.
4. No harassment, abuse, or unnecessary drama.

## Enforcement

If you break these rules, maintainers may remove content, close issues or pull requests, or revoke participation in the project.

That’s it.
